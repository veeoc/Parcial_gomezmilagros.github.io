Preguntas de análisis

1. ¿Se trata de una GUI? ¿De qué tipo?

Sí, este sitio web es una GUI (Graphical User Interface) — interfaz gráfica de usuario. Es de tipo web browser-based: el usuario interactúa con ella desde un navegador, usando elementos visuales como texto, imágenes e hipervínculos. A diferencia de interfaces de escritorio nativas, esta GUI es declarativa (HTML/CSS), multiplataforma y no requiere instalación. La interacción principal es la navegación mediante clics en enlaces y desplazamiento vertical.

2. ¿Qué permite y qué no permite la licencia CC BY-NC 4.0?

Este sitio está publicado bajo Creative Commons BY-NC 4.0 (Atribución – No Comercial). Permite: copiar, redistribuir y adaptar el material en cualquier medio, siempre que se dé crédito a la autora y se indique si hubo cambios.
