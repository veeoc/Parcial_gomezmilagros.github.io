Preguntas de análisis

1. ¿Se trata de una GUI? ¿De qué tipo?

Sí, este sitio web es una GUI (Graphical User Interface) — interfaz gráfica de usuario. Se trata de una GUI de tipo web browser-based: el usuario interactúa con ella a través de un navegador web, usando elementos visuales como texto, imágenes, hipervínculos y estructura jerárquica de contenidos. A diferencia de interfaces de escritorio nativas, esta GUI es declarativa (escrita en HTML/CSS) y se despliega en un entorno multiplataforma. No requiere instalación; la interacción principal es la navegación mediante clics en enlaces.
